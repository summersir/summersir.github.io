最近在查找关于表单的验证，发现了这个轻量级的轻量级的JavaScript表单验证，字符串验证validator.js。
按照他Github上所说的：轻量级的，没有依赖，支持UMD，~3kb。的确不大看着很好理解。和大家分享一下。

安装使用

模块

在应用中引用 validator.min.js 文件

$ npm install validator.tool --save

客户端使用

在应用中引用 validator.min.js 文件

<script type="text/javascript" src="dist/validator.min.js"></script>

应用在表单中的方法。

说明文档

new Validator(formName, option, callback)

formName

formName 是标签中<form> 中的 id 或者 name 的值，如上面的example_form

option

  • name -> input 中 name 对应的值
  • display -> 验证错误要提示的文字
  • rules -> 一个或多个规则(中间用|间隔)
    • is_email -> 验证合法邮箱
    • is_ip -> 验证合法 ip 地址
    • is_fax -> 验证传真
    • is_tel -> 验证座机
    • is_phone -> 验证手机
    • s_url -> 验证URL
    • required -> 是否为必填
    • max_length -> 最大字符长度
    • min_length -> 最小字符长度
